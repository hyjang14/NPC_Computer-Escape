#### 🎮 게임 설명 🎮

과연 파이는 현실로 돌아갈 수 있을까?

- 컴퓨터 속을 탐험하고, 현실로 돌아가기 위한 단서를 찾으세요.
- 낯선 npc와 친해져 탈출을 위한 정보를 얻으세요.
